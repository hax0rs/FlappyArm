About

Flappy Arm is a team based game in which a team of players controls joints on a robotic arm to cooperatively pick up and move objects. The aim of Flappy Arm is to inspire and educate younger students about the multi-disciplinary effort required to create similar real world robots and systems.

Technology and Components

The technology stack and components/design used with FlappyArm attempts to make the project as economically viable and beginner friendly as possible. The following points underline key features and the reason for their selection:

  • Open Source Arm Design - The project utilises the 7 Servo 6DOF Robotic Arm. The arm is extremely versatile and can be produced for around $50 from a multitude of materials (ply, mdf, plastic, etc.)
  • Raspberry PI - The Raspberry PI is designed to be a cheap, disposable and easy to use computer and thus FlappyArm uses it to lower the economic threshold to robotics and control.
  • Python 3 - A further advantage to using Python is that it comes pre installed on all Raspberry PI devices, removing the complex install process that can be daunting to beginners. Lastly using Python 3 allows for testing and development to be done rapidly, rather than having to wait for the program to compile or rewrite inbuilt functions in a language like C.
  • Flask Web Framework - The Flask webframework provides an easier entry into webdevelopment than other comparable Python frameworks such as Django which has higher levels of abstraction. Ideally, we believe Flask will expose young students to concepts such as routing and template rendering and not seem as daunting as other frameworks.
  • HTML/CSS - Using HTML and CSS provides an easier entry into GUI design and development as opposed to traditional GUI libraries such as Tcl/Tk and Qt which can be duanting as a beginner.
